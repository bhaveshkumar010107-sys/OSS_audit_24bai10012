## 📌 Project Overview

This project presents a detailed audit of **Node.js**, a widely used open-source JavaScript runtime environment. The main focus of this audit is to understand how Node.js works from both a technical and conceptual perspective.

The report explores its origin, licensing model, ethical aspects of open-source development, its behavior in a Linux environment, and how it compares with proprietary solutions offered by major technology companies.

Node.js plays a significant role in modern web development, especially in building fast, scalable, and efficient applications.

---

## 🎯 Objectives

The key objectives of this project are:

- To study the origin and evolution of Node.js  
- To understand its open-source license and underlying philosophy  
- To analyze how Node.js operates within a Linux environment  
- To explore the ecosystem of tools and frameworks built around Node.js  
- To compare Node.js with proprietary technologies developed by Big Tech companies

Each script demonstrates a specific concept or functionality using Node.js.

---

## ⚙️ Scripts Description

### 🔹 Script 1 – System Report  
This script uses the built-in `os` module to display basic system information such as the current user, operating system, and system uptime.

---

### 🔹 Script 2 – Package Checker  
This script checks whether a specific package (Express) is installed globally. It uses the `child_process` module to execute system-level commands.

---

### 🔹 Script 3 – Directory Analyzer  
This script reads and displays all files and folders present in the current directory using the `fs` module.

---

### 🔹 Script 4 – Log Analyzer  
This script reads a log file and counts how many times the word "error" appears. It demonstrates simple text processing and file handling.

---

### 🔹 Script 5 – Manifesto Generator  
This script creates a text file containing a short message about open-source philosophy. It shows how Node.js can be used for file creation and writing operations.
